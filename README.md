# simple-teapot
An http server believing it's a teapot

## Depedencies
- Node.js
- npm

## Install
- `git clone https://github.com/Grenadingue/simple-teapot`
- `cd simple-teapot`
- `npm install`

## Configure
- if needed update the `serverPort` value in `server.js` (default `8080`)

## Run
- `npm run start`
